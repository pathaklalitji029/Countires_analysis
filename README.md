# Countires_analysis
This project demonstrates data analysis and problem-solving skills using Pandas and NumPy.
The objective is to analyze a real-world dataset, perform data cleaning, manipulation, and answer analytical questions using efficient Pandas and NumPy operations.

The project focuses on extracting meaningful insights from structured data through exploratory data analysis (EDA).

Source: Kaggle (CSV format)

Type: Structured tabular data

Description:
The dataset contains multiple attributes such as numerical and categorical variables, allowing analysis through filtering, aggregation, and statistical operations.

Tools used:-
Python

Pandas – Data manipulation and analysis

NumPy – Numerical computations and array operations
